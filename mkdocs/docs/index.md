## Блоки кода

### Python с номерами строк
```python linenums="1"
def factorial(n):
    if n == 0:
        return 1
    return n * factorial(n - 1)

# Вычисляем факториал
result = factorial(5)
print(f"5! = {result}")
```

### Подсветка строк
```python hl_lines="2 3"
def hello(name):
    # Эта строка будет подсвечена
    message = f"Hello, {name}!"
    return message
```
