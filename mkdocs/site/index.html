        return 1
    return n * factorial(n - 1)

result = factorial(5)
print(f"5! = {result}")

      Клавиши¶

      Используйте следующие комбинации клавиш:

      • ++ctrl+c++ для копирования
      • ++ctrl+v++ для вставки
      • ++ctrl+alt+del++ для диспетчера задач
